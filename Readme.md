# Description

Debian image provided with utilities to deal with ec2, docker and kubernetes

Contains: 
* Awscli
* Kubectl
* Docker
* session-manager-plugin
* Awless
* SBT
* gimme-aws-creds
* openjdk-8-jdk, gradle
* python3-pip

## Usage

Just run<br/>
```shell 
vagrant up  
```
<br/>
